# Algorithms and Architectures CA.
Weather Data Analyzer using manual sorting, searching, merging, validation, and parallel merge sort in Python.( algorithms and architectures)
This project is a Python console application developed for COMM107J Algorithms & Architectures. It analyses weather datasets using manually implemented algorithms, including Bubble Sort, Insertion Sort, Merge Sort, Quick Sort, Linear Search, Binary Search, dataset merging, and depth-limited Parallel Merge Sort.

The aim of the project is to demonstrate algorithm design, time-complexity analysis, data validation, and clear console-based execution without relying on Python’s built-in sorting or searching shortcuts.
